WEB PROGRAMING

Course Code Course Title
BCSE203 Web Programing
Course Description
  • Introduces the fundamentals of web development and internet technologies
  • Covers basic building blocks such as HTML for structure and CSS for styling
  • Explains how to create interactive web pages using JavaScript
  • Provides knowledge of responsive design and modern web layouts
Eligibility: Undergraduate students from Computer Science
Syllabus Duration
Module 1: Introduction 2 Hours
World Wide Web evolution, Email, FTP, Cloud Computing, ISP, IP Address, URL, DNS, Browsers, Search Engines, Web Server vs Application Server.
Module 2: HTML 2 Hours
HTML tags, structure, lists, images, semantic elements, header, footer, navigation, articles and sections.
Module 3: CSS 2 Hours
CSS syntax, selectors, box model, margins, padding, borders, fonts, colors, hosting and Git basics.
